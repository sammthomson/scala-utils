Scala Utils
===========

Some utility functions I commonly find myself needing.


SBT
---

Add the following lines to your `build.sbt`:

```sbt
resolvers += Resolver.bintrayRepo("sammthomson", "maven")

libraryDependencies += "org.samthomson" %% "scala-utils" % "0.2"
```
